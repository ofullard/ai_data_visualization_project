# AI Chat + Data Visualization

This workspace contains:

- **frontend/**: React (Vite + TypeScript) chat UI with lightweight data visualization.
- **backend/**: Streamlit app that provides AI-styled responses and live analytics.

## Frontend

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## Backend (Streamlit)

1. Create a Python 3.12 environment.
2. Install dependencies: `pip install -r backend/requirements.txt`
3. Run the app: `streamlit run backend/app.py`

Replace the AI response logic in `backend/ai_client.py` with your preferred LLM provider when ready.
